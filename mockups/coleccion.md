# Vista: Colección

Dos pestañas. Aquí vive el leveleo de consejeros (gastar recursos).

## A — Pestaña Consejeros
```
+----------------------------------------------+
| < volver                             [*] opc |
|                  COLECCIÓN                   |
| [ CONSEJEROS ]                 [ GENERALES ] |
|----------------------------------------------|
| Consejeros            4 / 12                 |
| +--------+ +--------+ +--------+             |
| |[OFE]Lv3| |[DEF]Lv1| |[MAN]Lv2|             |
| | Sir A. | | Dame B.| | Lord C.|             |
| +--------+ +--------+ +--------+             |
| +--------+ + - - - -+ + - - - -+             |
| |[OFE]Lv1| |  ???   | |  ???   |             |
| | Ser D. | | bloq.  | | bloq.  |             |
| +--------+ + - - - -+ + - - - -+             |
|----------------------------------------------|
| tap consejero -> subir nivel (gasta $)       |
+----------------------------------------------+
```
- Grid de consejeros con stat afín y nivel. Slots bloqueados se desbloquean al obtener nuevos (eventos).
- Tap en un consejero → subir nivel (gasta recursos).

| +----------------------------------+         |

- Acciones: ver replay de su mejor batalla / mandarlo a la arena.
